# Baby Essentials

## Installation

1. Déposez tous ces fichiers dans votre repo GitHub (drag & drop)
2. Ajoutez `src/App.jsx` : copiez le code depuis l'artefact Claude "Baby Essentials v7"
3. Connectez le repo sur vercel.com → Deploy

## Structure
```
baby-essentials/
├── src/
│   ├── App.jsx      ← À ajouter manuellement depuis Claude
│   └── main.jsx
├── public/
│   └── favicon.svg
├── index.html
├── package.json
├── vite.config.js
└── vercel.json
```
